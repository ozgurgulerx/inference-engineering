# Templates

Reusable starting points for recurring note types inside `days02`.

## Files

- `daily-note.md` for day-by-day progress logs
- `experiment-card.md` for benchmark and profiling runs
- `section-summary.md` for consolidating durable takeaways

## Usage

- Start from a template instead of a blank file.
- Keep finished notes in the relevant section folder.
- Prefer editing a copy, not the template itself.
