# Daily Notes

This folder is the running log for active work inside `days02`.

## Purpose

- Capture what was attempted on a given date
- Record observations while they are still fresh
- Link deeper write-ups back to the day they were discovered

## Conventions

- Use one file per date in `YYYY-MM-DD.md` format.
- Keep entries short and concrete.
- Promote durable insights into the relevant numbered section once they stabilize.
