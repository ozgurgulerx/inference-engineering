## Minimum Evidence

Every experiment should record:

- GPU and memory.
- Container image.
- Model.
- Server command.
- Client command.
- Raw benchmark output.
- TTFT, total latency, throughput, and error rate.
- Cost notes.
- What changed in my understanding.
